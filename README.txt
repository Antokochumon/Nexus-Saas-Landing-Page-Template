== Description ==

Nexus is a modern, clean, and fully responsive HTML5 landing page template designed for SaaS products, startups, and software companies. It features a sleek design with glassmorphism, smooth animations, and both light/dark mode support.

== Features ==

- Fully responsive, mobile-first design
- Sticky navbar with animated hamburger and glassmorphism
- Animated mobile menu (slide-in from right, glass effect, icons)
- Light/Dark mode toggle with persistent theme and animated icons
- Hero section with Lottie animation, gradients, and call-to-action
- Clients/logos section with grayscale and hover effects
- Countdown timer for special offers
- Features section with glass cards, icons, and hover animations
- About/Why Us section with image highlights and feature list
- Stats section with animated counters
- Testimonials section powered by SwiperJS, glass cards, and star ratings
- Pricing section with monthly/yearly toggle, animated cards, and highlight for popular plan
- FAQ section with accordion and smooth expand/collapse
- Contact section with glass card, contact info, social icons, and validated form
- Footer with company info, quick links, resources, and contact details
- Back to top button with smooth scroll
- Modern animations using AOS (Animate On Scroll) and custom CSS
- Newsletter and contact forms with validation
- Well-organized, commented, and modular code
- SEO optimized and cross-browser compatible

== Installation ==

1. Unzip the downloaded package.
2. Upload the template files to your web server or hosting.
3. Open index.html in your browser to view your site.
4. Edit the HTML, CSS, and JS files as needed for your brand and content.

== Customization ==

- Change colors by editing Tailwind utility classes (e.g., blue-600, purple-600).
- Replace images by updating the <img> tags with your own URLs or files.
- Edit content directly in index.html. Each section is commented for easy navigation.
- Remove or comment out any section you do not need. All features are modular.

== JavaScript Features ==

- Theme toggle (remembers user preference, updates icons)
- Mobile menu (slide-in/out, disables background scroll, closes on link click)
- Smooth scroll for anchor links
- FAQ accordion (expand/collapse, ARIA accessibility)
- Pricing toggle (monthly/yearly)
- Newsletter & contact forms (client-side validation)
- SwiperJS for testimonials carousel
- AOS for section animations
- Lottie animation for hero section
- Back to top button (appears on scroll, smooth scroll)
- Countdown timer for special offers
- Animated stats counter (numbers animate when in view)
