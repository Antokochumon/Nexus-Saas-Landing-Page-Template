Nexus Documentation

A complete guide to install, customize, and use your Nexus SaaS Landing Page Template

Introduction

Thank you for choosing Nexus! This documentation will help you install, customize, and launch your SaaS landing page quickly and easily.

  • Fully responsive, modern, and animated design
  • Light/Dark mode support with smooth toggle
  • Easy customization with Tailwind CSS utility classes
  • Well-commented, organized, and modular code
  • Glassmorphism, gradients, and modern UI effects
  • Animated mobile menu with slide-in and glass effect
  • Hero, Features, About, Stats, Testimonials, Pricing, FAQ, Contact, and Footer sections
  • Integrated SwiperJS for testimonials, AOS for scroll animations, and Lottie for hero animation
  • Newsletter and contact forms with validation
  • Countdown timer, animated stats, and back-to-top button

File Structure

Nexus SaaS Landing Page Template/
├── index.html
├── documentation.html

├── style.css

All main files are in the root. CSS, JS, and images are organized in their respective folders.

Installation

  1. Unzip the downloaded package.
  2. Upload the template files to your web server or hosting.
  3. Open index.html in your browser to view your site.
  4. Edit the HTML, CSS, and JS files as needed for your brand and content.

Customization

Changing Colors

Nexus uses Tailwind CSS utility classes. To change the primary color, search for classes like blue-600 or purple-600 and replace them with your preferred color.

Updating Content

Edit the content directly in the index.html file. Each section is clearly commented for easy navigation.

Changing Images

Replace image URLs in <img> tags with your own images. Place your images in the assets/images/ folder for better organization.

Enabling/Disabling Features

You can easily remove or comment out any section you do not need. All features (testimonials, pricing, FAQ, etc.) are modular and can be edited or removed as desired.

JavaScript Features

  • Theme Toggle: Remembers user preference and updates icons
  • Mobile Menu: Slide-in/out animation, disables background scroll, closes on link click
  • Smooth Scroll: For all anchor links
  • FAQ Accordion: Expand/collapse with ARIA accessibility
  • Pricing Toggle: Switches prices between monthly and yearly
  • Newsletter & Contact Forms: Client-side validation and feedback
  • SwiperJS: For testimonials carousel
  • AOS: Animate On Scroll for section reveals
  • Lottie Animation: For hero section
  • Back to Top Button: Appears on scroll, smooth scroll to top
  • Countdown Timer: For special offers
  • Animated Stats Counter: Animates numbers when in view
